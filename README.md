# HEA Data Extraction using LLMs

This repository contains pipelines for extracting High-Entropy Alloy (HEA) data from scientific literature using large language models.

## Repository Structure

- **`q1_alloy_text_extraction/`**  
  Contains code and utilities for extracting alloy-related information from **textual content**.

- **`q2_alloy_table_extraction/`**  
  Contains code and utilities for extracting alloy-related information from **tables**.

Each folder is self-contained and corresponds to a specific stage of the overall extraction pipeline.
